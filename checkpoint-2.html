Level 1

Implement the following design of a feedback form. Try to use semantic HTML elements where applicable. It should be centered horizontally. It should use the background color #ffbab5. Bonus points for the gradient button!

In my solution I created a form element with the following structure; the form had a fieldset, which had a border set to none. Within this fieldset there was a legend describing the form, followed by labels that were set to hidden for the design, but still available for screenreaders for accessability, that describe the 3 following input elements, and 1 textarea element.

For the design in general I used flexbox to align the items up as needed, as well as using a linear gradient for the button styling as well as for the background behind the form.
